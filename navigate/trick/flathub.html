Cách tải flathub trên ubuntu
Nếu ai từng sử dụng ubuntu trên máy yếu và cài một số phân mềm qua store mặc định của ubuntu (snap) thì thấy nó rất lag hoặc nó sẽ khởi động rất lâu. Nhưng nêú ai từng thử flathub sẽ thấy rất nhiều packed từ flathub khắc phục được điều này. Vậy nên đây sẽ là hướng dẫn cách cài flathub cho ubuntu.

Yêu cầu

- Máy tính đang chạy ubuntu (điều đương nhiên)

Hướng dẫn

Bước 1: Mở terminal và copy đoạn code dưới đây dán vào

Bây giờ chúng ta sẽ cài flatpak
sudo apt install flatpak
sudo apt install gnome-software-plugin-flatpak

Bước 2: Khởi động lại, tiếp tục mở lại terminal và copy đoạn code dưới đây dán vào

Sau bước 1 các bạn kiểm tra thì sẽ thấy 1 ứng dụng mới tên là software, đó chính là store của flathub, ở đây nếu bạn muốn cài theo package snap thì vẫn có thể làm được bằng cách thay đổi package trước khi nhấn install. Bây giờ sẽ cài thêm flathub repository
flatpak remote-add --if-not-exists flathub https://dl.flathub.org/repo/flathub.flatpakrepo

Cuối cùng hãy tiến hành khởi động lại 1 lần nữa là xong
